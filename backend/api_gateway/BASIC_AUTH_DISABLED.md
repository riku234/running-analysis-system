# Basic認証について

## 概要

本システムでは、Basic認証が有効になっています。外部からアクセスする際は、認証情報が必要です。

## 認証情報

**ユーザー名**: `xebio`  
**パスワード**: `20251001`

詳細は `BASIC_AUTH_CREDENTIALS.md` を参照してください。

## アクセス方法

### ブラウザでのアクセス
1. `http://54.206.3.155/` にアクセス
2. 認証ダイアログが表示されたら、認証情報を入力

### curlでのアクセス
```bash
curl -u xebio:20251001 http://54.206.3.155/
```

## 確認方法

```bash
# 認証なし（401エラーが返ることを確認）
curl http://54.206.3.155/

# 認証あり（HTMLが返ることを確認）
curl -u xebio:PASSWORD http://54.206.3.155/
```

## 注意事項

- Basic認証はセキュリティのため有効化されています
- 認証情報は適切に管理してください
- パスワードを忘れた場合は、新しいパスワードを設定する必要があります
